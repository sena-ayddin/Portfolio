Showcase / Web Development Project

Personal Portfolio Website

A personal portfolio website created as part of a university course to present academic and technical work in a structured and visually clean format.

Goal

To design and develop a responsive and minimal portfolio that showcases projects and academic progress. The site was designed first in Figma as a wireframe and hi-fi prototype, then coded from scratch.

Tech Stack

HTML CSS JavaScript Figma

Features

Smooth scroll navigation, fade-in scroll animations, interactive contact form with DOM manipulation, responsive layout, and a clean design system built around a burgundy and pink color palette.

What I Learned

Frontend development basics, UI/UX design principles, responsive layout structuring, JavaScript DOM manipulation, and the full process of designing and building a multi-section website from scratch.
