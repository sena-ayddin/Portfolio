## 📋 Project Overview
A minimal and elegant portfolio website presenting my academic background, projects, and skills as a Computer Engineering student. This version includes JavaScript interactivity, a contact form, and scroll animations.

## 🤖 AI Usage
AI assistance was used in the following parts:
- **Color palette:** I knew I wanted warm burgundy and pink tones but was unsure which exact hex codes would work well together — AI helped me explore options and I selected the ones that matched my vision
- **Typography:** I wanted a classic, elegant font combination — AI suggested several options and I chose Playfair Display + Lora
- **Code optimization:** Some repetitive parts of the CSS and JavaScript were written with AI assistance
- All design decisions, content, and structure were made by me

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (DOM Manipulation, IntersectionObserver)
- Figma (Lo-fi Wireframe + Hi-fi Prototype)

## ✨ Features
- Smooth scroll navigation
- Fade-in scroll animations (IntersectionObserver)
- Interactive contact form with DOM manipulation
- Multi-page project detail pages
- Sticky navigation bar
- Hover effects on cards and buttons

## 🎨 Design Decisions
- **Color palette:** Burgundy (#7D1535) and pink (#E8547A) — warm, elegant, personal
- **Typography:** Playfair Display (headings) + Lora (body) — classic and readable
- **Layout:** Dark hero section with alternating light/dark sections inspired by Susan Allen's portfolio
